## Attempt from server
This is an example dashboard from Tableau which I have sized for the website and uploaded to our server. Access is restricted to Analysts. If you are logged in you will be able to see it. If you aren't logged in, you will be prompted to do so with a pop-up log in screen.

Dashboard size is width 600 x height 850. This can be adjusted (e.g. reduce height). You would just need to adjust the frame size when adding link into the blog. You can do this by trial-and-error if you need to. It usually only takes a minute for the page to go live so you can adjust/go-live/adjust/go-live...etc until you have it looking like you want.

Potentially, wider dashboards could be included, we'd just need to remove the boxes on the right of the page (which aren't needed anyway) and also check what happens when these are viewed on a phone, etc. Might take me a bit longer to figure that one out though.
